== Description ==

### DIVI WOOCOMMERCE MENU

A Divi Plugin to display WooCommerce products as a mega menu. Each module works with the children modules structure (Like the accordion default module provided by Divi)
 what it becomes this module a very powerful, because you can display different woocommerce product queries in each module with customized design settings on each one.
 Divi WooCommerce menu is fully compatible with the latest Divi Theme and plugin, and it works smoothly with the recently released Divi theme builder.

== Installation ==

Automatic installation is the easiest option as WordPress handles the file transfers itself and you do not need to leave your web browser. To do an automatic install of DIVI Enhancer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

### How to use

Insert the PBC Woo Module in your builder. The first option you will is the plus icon to add menu items. Each item added should be entirely adjusted inside of the module children, so if you want to change the item name, background color, product titles, etc must be do it here.
The menu settings, like menu items background, alignment and general options, must be setted on the main module.

### Is Divi WooCommerce Menu responsive?
Yes, is completely responsive. In fact, it has an option to adjust the setting even if you works with the browser inspector and changes constantly the window size.
